# Bonus Card Usage Analysis 🛒

## Objective
Determine what proportion of customers who purchased 3 or more products used a bonus card — and what this means for customer loyalty in Azerbaijani retail.

## Dataset
Retail transaction data from an Azerbaijani store chain, containing customer purchase records, product counts, and bonus card usage flags.

## Methods
- Data cleaning: handled missing values, removed duplicates, standardized formats
- Filtered transactions to customers with 3+ product purchases
- Calculated bonus card usage rate within this customer segment
- Exploratory data analysis (EDA) with summary statistics and group comparisons

## Key Finding
**~49.8% of customers who bought 3+ products used a bonus card**, suggesting that loyal, high-volume shoppers are almost equally split between card users and non-users — indicating significant untapped potential in the loyalty program.

## Tools & Technologies
- **Language:** R
- **Libraries:** dplyr, ggplot2
- **Output:** CSV results file with aggregated findings
